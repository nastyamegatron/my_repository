Тема: Анализ активности в интернете

Этап 1:
Написать программу или плагин для браузера, 
собирающую информацию об активности пользователя в браузере. 
Выполняется регистрация описанных в методичке событий. 
Можно использовать готовую прорамму (искать в интернете по запросу keylogger).

Этап 2:
Создать выборку, состоящую из 50 наборов данных для каждого исследуемого вида активности. 
Набор данных – это группа из 4 чисел – количество перечисленных действий пользователя за минуту. 
Таким образом, выборка будет состоять из 5*50 = 250 наборов данных (см. таблицу в методичке). 
По 200 записям из этой таблицы (обучающая выборка) обучить классификатор. 
По оставшимся 50 – провести тестирование классификатора и оценить, насколько точно он определяет вид деятельности пользователя.

Этап 3:
Реализовать режим записи событий. 
По окончании записи событий программа выполняет классификацию 
и выдает ответ о том, чем занимался пользователь.
